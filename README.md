# InAndOutTelegramBot
 Telegram bot to keep track of income and expenses.
 Its commands are: 
### /start and /help
It gives information about the InAndOut telegram bot.
### /add
It will store a transaction with a specific category, amount, date and description on Google Sheets.
### /removeLast
Function that remove the last in or out Transaction.
### /monthResume 
The user can choose a month, then he will have a resume of all in and outs of that specific month.
### /yearResume 
Resume of all in and outs of the current year.
